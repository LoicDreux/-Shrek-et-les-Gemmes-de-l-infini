Shrek et les Gemmes de l’infini est un FPS que j'ai fait dans Unity pour le projet final du cour de programmation 2. Le jeu a pour thème les memes absurdes, ainsi plusieurs univers sont combinés ensemble. Le joueur incarne Shrek et doit trouver les 6 Gemmes de l’infini avant le temps imparti en évitant le monstre, un chat avec une machette.

Pour plus d'informations, voir mon portofolio.
